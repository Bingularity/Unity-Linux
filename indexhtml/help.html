Remaster and Backup


You can create your remaster of any distribution based on Unity Linux with the script "mkremaster".

The Unity Logo


The Unity Linux logo is a tree leaf for a reason. We liken Unity Linux as the core or root of the distributions that build on it. Like a tree, each subsequent distribution based on Unity would be a 'Branch'. Branches are made easily with the LiveCD tools at the core of every Unity Linux based desktop.

Did you know?


Unity Linux packages are built using a 'chroot' which means that each package is built in a pristine environment so that developers are able to identify hard to find build requirements quickly and easily. It also affords developers the ability to build 32bit packages on a 64bit host.


Need help installing a desktop? Get up and running quickly on your favorite desktop. Just click on the icon of your favorite desktop below and the Smart Package Manager will begin to install it.

                



Support Resources


  • Community Forums - The community forums are a vast resource of help for users. Need help installing a desktop? Need help remastering or branching into your own distribution? Looking for help packaging? This is a 'first stop' for users to find their way.

  • Mailing List - Our developers have an active mailing list where they talk about upcoming features, packaging, troubleshooting, and tips/tricks. The mailing list is an excellent place to understand how Unity developers work and to interact with them as they continue to develop Unity Linux.

  • Documentation Wiki - Our documentation wiki is geared to help you find out more about the Unity Linux project, how it is organized and financed, who is a part of it, and how you can do more with Unity Linux. This community developed wiki is an excellent resource for general knowledge about Unity Linux.

  • Internet Relay Chat - Looking for Unity Linux on IRC? We have a channel on Freenode: #unitylinux. You can join us there using your favorite IRC client or you can use the link provided at the start of this information to open a page that will connect you in your browser. Please note that not all developers use IRC... so please be patient with your questions!
